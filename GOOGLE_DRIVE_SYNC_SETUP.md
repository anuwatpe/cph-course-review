# ตั้งค่า Google Drive Sync สำหรับแดชบอร์ดรายวิชา

ไฟล์ `google-drive-sheet-sync.gs` ใช้กับ Google Apps Script ใน Google Sheet ที่เว็บนี้ดึงข้อมูลอยู่
เพื่อสแกนไฟล์ใน Google Drive แล้วเติมค่าใน Sheet อัตโนมัติ

## คอลัมน์ที่สคริปต์ใช้

ต้องมีคอลัมน์รหัสวิชาอย่างน้อยหนึ่งชื่อจากนี้:

- `รหัสวิชา`
- `รหัส`
- `Course Code`
- `course_code`

สคริปต์จะสร้าง/อัปเดตคอลัมน์เหล่านี้ให้เองถ้ายังไม่มี:

- `ลิงก์ไฟล์`
- `อัพไฟล์`
- `Google Drive File ID`
- `ชื่อไฟล์ล่าสุด`
- `วันที่พบไฟล์`

สคริปต์จะสร้าง/อัปเดตคอลัมน์เหล่านี้สำหรับโฟลเดอร์รายวิชา:

- `Drive Folder ID`
- `ลิงก์โฟลเดอร์อัปโหลด`
- `ลิงก์โฟลเดอร์เทอม`
- `แชร์โฟลเดอร์ให้`

ถ้าคอลัมน์ `Drive Folder ID` ยังว่าง สคริปต์สามารถสร้างโฟลเดอร์ของรายวิชานั้นให้เองได้

ถ้าต้องการให้อาจารย์แต่ละวิชาอัปโหลดไฟล์เอง ให้เพิ่มคอลัมน์อีเมลอย่างน้อยหนึ่งชื่อจากนี้:

- `อีเมล`
- `อีเมลผู้รับผิดชอบ`
- `อีเมลอาจารย์`
- `Instructor Email`
- `teacher_email`

ใส่ได้มากกว่าหนึ่งอีเมลในช่องเดียว เช่น `teacher1@school.ac.th, teacher2@school.ac.th`

## สร้างโฟลเดอร์รายวิชาอัตโนมัติ

ใน `google-drive-sheet-sync.gs` มีค่านี้:

```js
COURSE_ROOT_FOLDER_ID: '',
COURSE_ROOT_FOLDER_NAME: 'Course Review Uploads',
AUTO_CREATE_COURSE_FOLDERS: true,
GROUP_COURSE_FOLDERS_BY_TERM: true,
MAKE_FILES_VIEWABLE_BY_LINK: true,
AUTO_SHARE_COURSE_FOLDERS_WITH_INSTRUCTORS: false,
```

ถ้า `COURSE_ROOT_FOLDER_ID` ว่าง สคริปต์จะสร้างโฟลเดอร์แม่ชื่อ `Course Review Uploads`
ใน My Drive ให้อัตโนมัติ แล้วสร้างโฟลเดอร์แยกตามเทอมและรายวิชาใน Sheet

ถ้าต้องการใช้โฟลเดอร์แม่ที่มีอยู่แล้ว ให้ใส่ Folder ID หรือ URL ของโฟลเดอร์แม่ใน `COURSE_ROOT_FOLDER_ID`

โครงสร้างเริ่มต้นจะเป็น:

- `Course Review Uploads`
- `Course Review Uploads/2569_T1`
- `Course Review Uploads/2569_T1/412301_ชื่อรายวิชา`

ชื่อโฟลเดอร์เทอมสร้างจาก `ปีการศึกษา` และ `ภาคการศึกษา`
ชื่อโฟลเดอร์รายวิชาสร้างจาก `รหัสวิชา` และ `ชื่อรายวิชา`
เช่น:

- `2569_T1`
- `412301_ชื่อรายวิชา`

ถ้าไม่ต้องการแยกรายเทอม ให้เปลี่ยน `GROUP_COURSE_FOLDERS_BY_TERM` เป็น `false`

## คำแนะนำในการจัดการระยะยาว

แนะนำให้ใช้โครงสร้างรายเทอม เพราะเมื่อมีข้อมูลหลายปี โฟลเดอร์แม่จะยังอ่านง่าย:

- `Course Review Uploads/2569_T1/รายวิชา`
- `Course Review Uploads/2569_T2/รายวิชา`
- `Course Review Uploads/2570_T1/รายวิชา`

ถ้าสร้างโฟลเดอร์แบบเดิมไปแล้วและต้องการย้ายเข้าเทอม ให้ใช้เมนู:

- `Drive Sync` > `จัดโฟลเดอร์เดิมเข้าเทอม`

เมนูนี้จะย้ายเฉพาะโฟลเดอร์ที่มี `Drive Folder ID` อยู่ใน Sheet ตาม `ปีการศึกษา` และ `ภาคการศึกษา`

## สิทธิ์การเข้าถึงไฟล์

ถ้าลิงก์ไฟล์แสดงในเว็บแล้ว แต่เปิดไม่ได้หรือขึ้นว่าเป็น private แปลว่าไฟล์ใน Google Drive ยังไม่ได้เปิดสิทธิ์ให้ผู้ชมเว็บเข้าถึง

ไม่จำเป็นต้องเปิดทั้งโฟลเดอร์ `Course Review Uploads` เป็นสาธารณะเสมอไป ทางที่คุมง่ายกว่าคือเปิดสิทธิ์เฉพาะไฟล์ที่ถูกนำไปแสดงบนเว็บเป็น `Anyone with the link can view`

ค่าเริ่มต้นในสคริปต์ตอนนี้คือ:

```js
MAKE_FILES_VIEWABLE_BY_LINK: true,
MAKE_FOLDERS_VIEWABLE_BY_LINK: false,
```

หมายความว่า:

- ไฟล์ที่สคริปต์สแกนเจอและเติมลง `ลิงก์ไฟล์` จะถูกตั้งสิทธิ์ให้เปิดผ่านลิงก์ได้
- โฟลเดอร์รายวิชาและโฟลเดอร์แม่ยังไม่ถูกเปิดเป็นสาธารณะ
- ถ้าไฟล์มีข้อมูลที่ไม่ควรเปิดสาธารณะ ให้เปลี่ยน `MAKE_FILES_VIEWABLE_BY_LINK` เป็น `false`

ถ้ามีไฟล์ที่ถูกลิงก์ไปแล้วก่อนหน้านี้และยัง private อยู่ ให้ใช้เมนู:

- `Drive Sync` > `ตั้งสิทธิ์ไฟล์ให้เปิดผ่านลิงก์`

## ให้อาจารย์อัปโหลดไฟล์ในโฟลเดอร์รายวิชา

โฟลเดอร์ `Course Review Uploads` และโฟลเดอร์รายวิชาไม่ควรเปิดเป็นสาธารณะเพื่อให้อัปโหลด เพราะคนที่มีสิทธิ์แก้ไขโฟลเดอร์อาจลบหรือย้ายไฟล์ได้

แนวทางที่แนะนำคือ:

- เปิดดูไฟล์บนเว็บ: ใช้ `MAKE_FILES_VIEWABLE_BY_LINK: true`
- ให้อัปโหลดไฟล์: แชร์เฉพาะโฟลเดอร์รายวิชาให้กับอีเมลอาจารย์เจ้าของวิชา
- ไม่เปิดโฟลเดอร์แม่ `Course Review Uploads` เป็นสาธารณะ

ขั้นตอน:

1. ใช้คอลัมน์ `อีเมล` ใน Sheet
2. ใส่อีเมลอาจารย์ในแถวรายวิชาของท่านนั้น
3. ใช้เมนู `Drive Sync` > `แชร์โฟลเดอร์ให้อาจารย์`
4. ส่ง `ลิงก์โฟลเดอร์อัปโหลด` ให้อาจารย์ หรือให้อาจารย์เปิดจาก Sheet ถ้ามีสิทธิ์เข้าถึง Sheet

ค่าเริ่มต้นในสคริปต์คือ:

```js
AUTO_SHARE_COURSE_FOLDERS_WITH_INSTRUCTORS: false,
MAKE_FOLDERS_VIEWABLE_BY_LINK: false,
```

ค่าเริ่มต้นตั้ง `AUTO_SHARE_COURSE_FOLDERS_WITH_INSTRUCTORS` เป็น `false` เพื่อป้องกันไม่ให้ trigger ส่งอีเมลแชร์โฟลเดอร์ซ้ำ ๆ ให้อาจารย์
เมื่อต้องการแชร์โฟลเดอร์ ให้อัปเดตคอลัมน์ `อีเมล` แล้วกดเมนู `Drive Sync` > `แชร์โฟลเดอร์ให้อาจารย์` ด้วยตัวเอง

ข้อควรทราบ: สิทธิ์อัปโหลดใน Google Drive ต้องเป็นสิทธิ์ editor ของโฟลเดอร์ ซึ่งแปลว่าอาจารย์มีสิทธิ์จัดการไฟล์ในโฟลเดอร์รายวิชานั้นด้วย ถ้าต้องการป้องกันการลบไฟล์อย่างเข้มงวดกว่า ควรใช้ Google Form แบบ File Upload หรือใช้ Shared Drive ที่กำหนด role แยกละเอียดกว่า

## คำอธิบายเมนู Drive Sync

หลังติดตั้งสคริปต์และ reload Google Sheet จะมีเมนู `Drive Sync` เพิ่มขึ้นมา

### สร้างโฟลเดอร์รายวิชา

ใช้เมื่อเริ่มตั้งค่าระบบ หรือเมื่อเพิ่มรายวิชาใหม่ใน Sheet

เมนูนี้จะ:

- สร้างโฟลเดอร์แม่ `Course Review Uploads` ถ้ายังไม่มี
- สร้างโฟลเดอร์เทอม เช่น `2569_T1`
- สร้างโฟลเดอร์รายวิชา เช่น `412301_ชื่อรายวิชา`
- เติม `Drive Folder ID`
- เติม `ลิงก์โฟลเดอร์อัปโหลด`
- เติม `ลิงก์โฟลเดอร์เทอม`
- แชร์โฟลเดอร์ให้อีเมลอาจารย์ ถ้ามีคอลัมน์ `อีเมล`

เมนูนี้ยังไม่ดึงลิงก์ไฟล์ มคอ. มาใส่ใน `ลิงก์ไฟล์` ถ้าต้องการทำทั้งสร้างโฟลเดอร์และดึงไฟล์ ให้ใช้เมนู `สร้างโฟลเดอร์และสแกนไฟล์`

### จัดโฟลเดอร์เดิมเข้าเทอม

ใช้เมื่อเคยสร้างโฟลเดอร์รายวิชาไว้ใต้ `Course Review Uploads` แล้วต้องการย้ายให้เป็นระเบียบตามเทอม

เมนูนี้จะ:

- อ่าน `Drive Folder ID` ของแต่ละรายวิชาใน Sheet
- ดู `ปีการศึกษา` และ `ภาคการศึกษา` ของแถวนั้น
- สร้างโฟลเดอร์เทอมถ้ายังไม่มี
- ย้ายโฟลเดอร์รายวิชาเข้าไปอยู่ใต้โฟลเดอร์เทอม
- อัปเดต `ลิงก์โฟลเดอร์อัปโหลด` และ `ลิงก์โฟลเดอร์เทอม`

เมนูนี้ไม่ลบไฟล์และไม่เปลี่ยนลิงก์ไฟล์เดิม แต่จะย้ายตำแหน่งโฟลเดอร์ใน Google Drive

### แชร์โฟลเดอร์ให้อาจารย์

ใช้เมื่อเพิ่มหรือแก้ไขอีเมลอาจารย์ใน Sheet แล้วต้องการให้อาจารย์มีสิทธิ์อัปโหลดไฟล์ในโฟลเดอร์รายวิชาของตัวเอง

เมนูนี้จะ:

- อ่าน `Drive Folder ID` ของแต่ละรายวิชา
- อ่านอีเมลจากคอลัมน์ `อีเมล` หรือชื่อคอลัมน์อีเมลที่รองรับ
- เพิ่มอีเมลเหล่านั้นเป็น editor ของโฟลเดอร์รายวิชา
- เติมคอลัมน์ `แชร์โฟลเดอร์ให้`

เมนูนี้ไม่เปิดโฟลเดอร์เป็นสาธารณะ และไม่แชร์โฟลเดอร์แม่ `Course Review Uploads`
Google Drive อาจส่งอีเมลแจ้งเตือนเมื่อเพิ่ม editor ใหม่ ดังนั้นควรกดเมนูนี้เฉพาะตอนเริ่มแชร์ครั้งแรกหรือเมื่อเพิ่มอีเมลใหม่เท่านั้น

### สแกนไฟล์และเติมลิงก์

ใช้เมื่อมีการอัปโหลดไฟล์ลงในโฟลเดอร์รายวิชาแล้ว และต้องการให้ Sheet อัปเดตลิงก์ไฟล์

เมนูนี้จะ:

- เปิดโฟลเดอร์จาก `Drive Folder ID`
- หาไฟล์ล่าสุดในโฟลเดอร์รายวิชานั้น
- เติม `ลิงก์ไฟล์`
- ตั้งค่า `อัพไฟล์` เป็น `TRUE`
- เติม `Google Drive File ID`
- เติม `ชื่อไฟล์ล่าสุด`
- เติม `วันที่พบไฟล์`
- ตั้งสิทธิ์ไฟล์ให้เปิดผ่านลิงก์ได้ เมื่อ `MAKE_FILES_VIEWABLE_BY_LINK` เป็น `true`

ถ้าแถวยังไม่มี `Drive Folder ID` และ `AUTO_CREATE_COURSE_FOLDERS` เป็น `true` สคริปต์จะสร้างโฟลเดอร์รายวิชาให้ก่อน แล้วค่อยสแกนไฟล์

### ตั้งสิทธิ์ไฟล์ให้เปิดผ่านลิงก์

ใช้เมื่อเว็บมีปุ่มลิงก์ไฟล์แล้ว แต่ผู้ใช้เปิดไฟล์ไม่ได้เพราะไฟล์ยังเป็น private

เมนูนี้จะ:

- อ่าน `Google Drive File ID` หรือ `ลิงก์ไฟล์` จากแต่ละแถวใน Sheet
- ตั้งสิทธิ์ไฟล์เป็น `Anyone with the link can view`
- ข้ามแถวที่ยังไม่มีไฟล์หรือลิงก์ไฟล์

เมนูนี้เหมาะสำหรับแก้ไฟล์ย้อนหลัง ส่วนไฟล์ใหม่ในอนาคตจะถูกตั้งสิทธิ์อัตโนมัติเมื่อ `MAKE_FILES_VIEWABLE_BY_LINK` เป็น `true`

### สร้างโฟลเดอร์และสแกนไฟล์

ใช้เป็นเมนูหลักสำหรับการทำงานประจำ เพราะรวมสองขั้นตอนในครั้งเดียว

เมนูนี้จะ:

- สร้างหรือตรวจโฟลเดอร์รายวิชา
- เติมลิงก์โฟลเดอร์ใน Sheet
- สแกนไฟล์ล่าสุดในแต่ละโฟลเดอร์
- เติมลิงก์ไฟล์และสถานะอัปโหลด

แนะนำให้ใช้เมนูนี้หลังเพิ่มรายวิชาใหม่ หรือหลังอัปโหลดไฟล์หลายวิชาพร้อมกัน

### สร้าง Trigger ทุก 5 นาที

ใช้เมื่อต้องการให้ระบบทำงานอัตโนมัติ

เมนูนี้จะตั้งเวลาให้รัน `syncDriveLinksToSheet` ทุก 5 นาที เพื่อสแกนไฟล์และเติมลิงก์ลง Sheet โดยไม่ต้องกดเอง

หมายเหตุ:

- เมนูนี้สร้าง trigger เพียงครั้งเดียว ถ้ามีอยู่แล้วจะไม่สร้างซ้ำ
- Trigger จะสแกนไฟล์และเติมลิงก์ แต่ไม่ได้ย้ายโฟลเดอร์เดิมเข้าเทอมทุกครั้ง
- Trigger จะไม่แชร์โฟลเดอร์ให้อาจารย์เมื่อ `AUTO_SHARE_COURSE_FOLDERS_WITH_INSTRUCTORS` เป็น `false`
- ถ้ามีการเพิ่มรายวิชาใหม่ trigger จะสร้างโฟลเดอร์ให้เองเมื่อ `AUTO_CREATE_COURSE_FOLDERS` เป็น `true`

## ต้องตั้งชื่อไฟล์เฉพาะไหม

ถ้าใช้โฟลเดอร์รายวิชาที่สคริปต์สร้างให้: ไม่จำเป็นต้องตั้งชื่อไฟล์เฉพาะมากนัก
เพราะสคริปต์จะใช้ไฟล์ล่าสุดในโฟลเดอร์ของรายวิชานั้น

ถ้าใช้โฟลเดอร์กลางรวมหลายรายวิชาแทน: ควรให้ชื่อไฟล์มีรหัสวิชาแบบชัดเจน เช่น

- `412301_มคอ3_ชื่อรายวิชา.pdf`
- `412301-มคอ4-ชื่อรายวิชา.docx`
- `PH412301_มคอ3_ชื่อรายวิชา.pdf`

หลักคือรหัสในชื่อไฟล์ต้องตรงกับคอลัมน์ `รหัสวิชา` หลังตัดช่องว่าง ขีดกลาง `/` และ `_`

## วิธีใช้งาน

1. เปิด Google Sheet ที่เว็บนี้ใช้เป็นแหล่งข้อมูล
2. ไปที่ `Extensions` > `Apps Script`
3. วางโค้ดจาก `google-drive-sheet-sync.gs`
4. ถ้ามีชื่อชีตเฉพาะ ให้ใส่ใน `SHEET_NAME`
5. ถ้าต้องการให้โฟลเดอร์รายวิชาอยู่ใต้โฟลเดอร์แม่ที่มีอยู่แล้ว ให้ใส่ `COURSE_ROOT_FOLDER_ID`
6. กด Run ฟังก์ชัน `createCourseFoldersFromSheet` หนึ่งครั้งเพื่ออนุญาตสิทธิ์
7. Reload Google Sheet แล้วใช้เมนู `Drive Sync` > `สร้างโฟลเดอร์และสแกนไฟล์`
8. ใช้เมนู `Drive Sync` > `สร้าง Trigger ทุก 5 นาที`

เว็บ `index.html` จะดึงข้อมูลจาก Sheet ทุก 5 นาทีอยู่แล้ว
และตอนนี้ถ้าแถวใดมี `ลิงก์ไฟล์` เว็บจะนับว่าอัปโหลดแล้วทันที
